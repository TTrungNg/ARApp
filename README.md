# AR Shopping app like iKEA Place 
Based on an Udemy course: [https://www.udemy.com/course/flutter-ios-android-augmented-reality-ar-ikea-clone-2023/](url)

## What I used
- Dart/Flutter
- Firebase
- Removing Background API
